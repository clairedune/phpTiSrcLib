convert a rgb ressource image into a greyscale image 

\-This function uses the following naive algorithm. \-For all the pixels of the image, compute the average values of the r,g,b componants and use it to set the destination pixel values \-:


\begin{DoxyCode}
 X = (R+G+B)/3
\end{DoxyCode}
